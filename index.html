Heat Map

Este mapa de calor muestra la concentración de vuelos según su horario, facilitando la identificación visual de las franjas con mayor o menor actividad.

Bar Chart

El gráfico de barras compara de manera directa los porcentajes de retraso entre diferentes categorías, permitiendo identificar rápidamente los extremos del rendimiento.

Bump Chart

Este Bump Chart permite hacer un seguimiento de la evolución de las clasificaciones a lo largo de un período, resaltando qué elementos mejoran o empeoran su posición.

Dashboard

Este Dashboard integrador proporciona una vista general combinando múltiples métricas. Interactúa con los elementos del gráfico para explorar los datos a mayor profundidad.
